== Description ==

Kashiwazaki SEO Hamburger Menuは、レスポンシブデザインに対応したハンバーガーメニュープラグインです。

**主な機能：**
* 指定したブレークポイント以下でハンバーガーメニューを表示
* メニュー位置のカスタマイズ（左、右、上、下）
* 既存のWordPressメニューを反映またはカスタムメニューを作成
* 色設定のカスタマイズ
* モバイルフレンドリーなUI/UX

**設定可能項目：**
* ブレークポイント（px）
* メニュー表示位置
* メニュータイプ（既存メニュー反映/カスタムメニュー）
* カスタムメニュー項目
* メニュー色
* 背景色

== Installation ==

1. プラグインファイルを `/wp-content/plugins/kashiwazaki-seo-hamburger-menu/` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューでプラグインを有効化
3. 「ハンバーガーメニュー」メニューから設定を行う

== Frequently Asked Questions ==

= どのようなメニューが表示されますか？ =

既存のWordPressメニューを反映するか、カスタムメニューを作成するかを選択できます。カスタムメニューの場合、メニュー名|URLの形式で設定できます。

= ブレークポイントはどのように設定すればよいですか？ =

一般的なモバイルデバイスの幅に合わせて設定することをお勧めします。例：768px（タブレット）、480px（スマートフォン）

= メニューの位置は変更できますか？ =

はい、左、右、上、下の4つの位置から選択できます。

= 色のカスタマイズは可能ですか？ =

はい、メニューの文字色と背景色をカスタマイズできます。
